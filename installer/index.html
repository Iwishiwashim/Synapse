Requirements

Synapse needs Python 3.10+ and Git. Install them if you haven't already.

Python 3.10+ — python.org/downloads
Tick "Add Python to PATH" during install on Windows.
Git — git-scm.com/downloads
Needed to clone the repo and for Synapse's auto-commit feature.

Verify in a terminal:

python --version git --version

API Keys

Synapse uses Gemini for embeddings and search. Only the Gemini key is required for daily use.

KeyUsed for
GEMINI_API_KEY required Everything — embeddings, search, import
aistudio.google.com/apikey (free)
GROQ_API_KEY optional Bulk export filtering only
console.groq.com (free tier)
OPENROUTER_API_KEY optional Bulk export filtering only
openrouter.ai/keys (free tier)

Add keys to a .env file in the Synapse folder:

GEMINI_API_KEY=your_key_here GROQ_API_KEY=your_key_here OPENROUTER_API_KEY=your_key_here

Install

Clone the repo, create a virtual environment, and install dependencies.

Windows (PowerShell):

git clone https://github.com/Santhosh-Stalin/Synapse.git cd Synapse python -m venv .venv .venv\Scripts\activate pip install -r requirements.txt

macOS / Linux:

git clone https://github.com/Santhosh-Stalin/Synapse.git cd Synapse python -m venv .venv source .venv/bin/activate pip install -r requirements.txt

Verify the install:

python -m pytest Diagnostics/test_core.py -v

Configure

Run setup to set your API key, vault path, write mode, and register Synapse with Claude Desktop and Claude Code.

python setup.py

What setup.py does:

  • Reads your Gemini API key from .env or prompts for it
  • Sets vault path (default: ./vault)
  • Asks for write mode — review (you approve each write) or auto (Claude writes directly)
  • Writes config.yaml
  • Updates Claude Desktop MCP config at ~/Library/Application Support/Claude/claude_desktop_config.json
  • Updates Claude Code MCP config
After setup, restart Claude Desktop. Synapse appears in the tools list automatically.
